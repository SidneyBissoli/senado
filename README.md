Access data from the Brazilian Federal Senate (Senado Federal) Open Data API from R.

## Installation

You can install the development version of senado from [GitHub](https://github.com/) with:

```r
# install.packages("pak")
pak::pak("SidneyBissoli/senado")
```

## Overview

`senado` provides a tidy interface to the [Senado Federal Open Data API](https://legis.senado.leg.br/dadosabertos/). All functions return tibbles with standardized `snake_case` column names.

**This package is under active development.**
